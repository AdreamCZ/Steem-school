Our team is happy for your motivation. To further aid you in your research on the subject, let us save you some time by introducing you to the best guides we were able to find. Choose one of these supported programming languages.

PYTHON

The official library : Steem-Python

Library that simplifies working with the Steem's blockchain and cryptographic technology : BEEM

RUBY

The official API : Steem-Ruby

API that adds usefull functions to the official core : Radiator

JavaScript

The official library: Steem-JS

Robust library that adds support for Node.js : DSteem

PHP

Unofficial collection of usefull tools for PHP. But is not quite production ready yet : Steem-tools

GO

Unofficial RPC client library for working with Steem using Golang : Steem-Go

Java

API wrapper to access data stored in the Steem blockchain using Java SteemJ
